Interface: Stellaris Colors


DESCRIPTION
-----------

[h1]Mod Description[/h1]
This mod changes the font color palette to the one used in Stellaris.
It has been designed with my two other mods in mind (see below) but can also be installed stand-alone.

[h1]Ironman[/h1]
This mod is Ironman compatible.

[h1]Additional Mods[/h1]
I recommend to subscribe to the following mods in addition to this mod:
[list]
[*] [url=http://steamcommunity.com/sharedfiles/filedetails/?id=735130352]Interface: Century Gothic[/url]
[*] [url=http://steamcommunity.com/sharedfiles/filedetails/?id=734350648]Tooltip: Century Gothic Bold 18[/url]
[/list]

[h1]Compatibility[/h1]
Apart from the font mods mentioned above this mod probably is not compatible with other font mods.

[h1]Miscellaneous[/h1]
If you are looking for a similar mod for Europa Universalis 4 I highly recommend the awesome [url=http://steamcommunity.com/sharedfiles/filedetails/?id=704256650]Stellaris UI Font[/url] by ross-g.
